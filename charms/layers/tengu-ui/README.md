# Overview

This Charm creates the [Tengu](http://tengu.intec.ugent.be) web ui. This web ui can be used to create hauchiwas end models on the Tengu platform.

# Usage

Install Charm.

    juju deploy cs:~tengu-bot/tengu-ui
